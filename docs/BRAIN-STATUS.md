## Confirmed high-priority backlog (verified twice)

1. **Multi-library grounding is untouched.** All 1,461 grounding
   formalizations resolve into Mathlib (1,424) + Init (37); FLT / Physlib /
   Cslib / Carleson / SpherePacking hold ~35k decls with ZERO concept
   coverage — the containers render but no concept links into them. Expect
   low QID yield (research-frontier decls often have no Wikidata item); the
   deliverable is as much "which frontier decls need QIDs minted" as
   "resolve existing QIDs". The next discovery fleet's target.
2. **Slogan licensing — RESOLVED 2026-07-04 by removal.** BRAIN.md:452 held
   TheoremGraph statement data to link-facts-only until its CC-BY-SA vs
   CC-BY-NC-SA contradiction is resolved; the build had baked
   theorem_matching formal_slogan into decl nodes and rendered it publicly.
   The spec's proposed fix (re-source from math-graph slogan.csv, CC-BY-4.0)
   turned out to be impossible — slogan.csv is INFORMAL-only (0 of 2.57M rows
   reference a formal statement id; another spec error). Decl slogans are now
   absent by policy; decl gloss = docstring + code (both cleanly licensed).
   Reinstate only if the theorem-matching license is clarified upstream
   (vilin@uw.edu) — or attach slogan.csv glosses to LITERATURE nodes, whose
   ids it does cover.
3. **Concepts D1 table + POST /api/concept (roadmap P2).** The human
   web-editing surface for the graph genuinely does not exist — /brain is
   read-only; AI edits flow through brain/proposals only. Precondition:
   the diffFields array-order fix (separate task already spawned).
4. **Standing decl-collision guard.** The Q3968/Q1000660 class (one decl,
   two QIDs, field-vs-object P31) was fixed once by the collision fleet but
   no BUILD-TIME check prevents regressions — belongs in test_acceptance.
5. **OpenAlex Topics cross-check.** Keyless 4-level taxonomy for every
   arXiv-linked paper — the independent placement signal for informal nodes
   the spec wanted; still entirely unused. The registry/schema slot exists.

Also real (medium): the brain rebuild is not wired into any nightly job
(site/ops regenerates /map only), and the atlas/map pipeline still reads
concept_graph v1 pending the promotion decision.
